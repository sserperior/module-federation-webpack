# Module Federation with Webpack Example

This sample consists of the 4 projects:

1. remote-app - front end app which exposes a form component (see remote-app/src/components/InsuranceForm.jsx).

2. remote-server - The Backend for Frontend (BFF) for remote-app.

3. host-app - front end app which consumes the form component exposed by remote-app.

4. host-server - The Backend for Frontend (BFF) for host-app.
